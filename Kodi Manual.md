# Kodi是什么

Kodi是一个播放器，可播放本地和网络上的视频、音乐、照片。能播放电视直播，可当作电视看。

# 怎么使用Kodi

1. 安装Kodi
2. 设置界面
3. 安装电视直播插件
4. 设置直播源
5. 播放电视直播
# 安装Kodi

以下安装和设置都是在Arch Linux系统上进行的。

- 安装Kodi： `yay -S kodi`

# 设置界面

首先设置字体，再将界面设置成中文。如果不先设置字体直接设置中文会导致界面乱码。

设置字体：
1. 打开Kodi进入首页。
2. 点击左上角的设置按钮（小齿轮）。
3. 点击`Interface`，进入设置界面。
4. 在设置页面左侧菜单找到`Skin`后点击。
5. 在`Skin`页面右侧找到`Fonts`，点击`Fonts`进入字体选择界面。
6. 在字体选择界面选择`Arial based`即可。

将界面设置成中文： 
1. 打开Kodi进入首页。
2. 点击左上角的设置按钮（小齿轮）。
3. 点击`Interface`，进入设置界面。
4. 在设置页面左侧菜单找到`Regional`后点击。
5. 在`Regional`页面右侧找到`Language`，点击`Language`进入选择语言界面。
6. 在选择语言界面选择`Chinese(Simple)`后界面就变成了中文。



# 安装电视直播插件

安装电视直播插件后可以在Kodi中直接看电视直播（需要在插件中配置直播源），就像一台真正的电视一样。

- 安装电视直播插件： `yay -S kodi-addon-pvr-iptvsimple`
- 重启Kodi

启用直播插件：
1. 打开Kodi进入首页。
2. 在首页左侧菜单中找到`插件`按钮，点击`插件`按钮进入插件管理页面。
3. 在插件管理页面左侧菜单中找到`我的插件`按钮，点击`我的插件`按钮进入我的插件管理页面。
4. 在我的插件管理页面中找到`PVR客户端`按钮，点击`PVR客户端`按钮进入PVR客户端管理页面。
5. 在PVR客户端管理页面找到刚才安装的电视直播插件： `IPTV Simple Client`，点击插件名字 `IPTV Simple Client`进入插件详细信息页面。
6. 在插件详细信息页面点击`启用`按钮即可启用该插件。

# 设置直播源

直播源就是一个电视直播的节目列表，需要配置到直播插件中才能看电视直播。

设置直播源：
1. 打开Kodi进入首页。
2. 在首页左侧菜单中找到`插件`按钮，点击`插件`按钮进入插件管理页面。
3. 在插件管理页面左侧菜单中找到`我的插件`按钮，点击`我的插件`按钮进入我的插件管理页面。
4. 在我的插件管理页面中找到`PVR客户端`按钮，点击`PVR客户端`按钮进入PVR客户端管理页面。
5. 在PVR客户端管理页面找到刚才安装的电视直播插件： `IPTV Simple Client`，点击插件名字 `IPTV Simple Client`进入插件详细信息页面。
6. 在插件详细信息页面点击`设置`按钮，进入设置页面。
7. 在设置页面点击`添加插件配置`按钮，进入添加插件页面。
8. 在插件添加页面中，点击`名称`按钮，会弹出名称输入界面，输入一个直播源的名字，该名字可随意输入，用来区分不同的直播源。输入完成后点击`确定`按钮即可。
9. 还是在插件添加页面中，继续点击`M3U播放列表URL`按钮，会弹出直播源地址输入界面，输入直播源地址： `https://live.zbds.top/tv/iptv4.m3u`（可以自行找其他直播源使用），输入完成后点击`确定`按钮即可。
10. 还是在插件添加页面中，点击`确定`按钮即可完成直播源地址的添加。

# 播放电视直播

设置完直播源地址之后，就可以播放电视直播节目：
1. 打开Kodi进入首页。
2. 在首页左侧菜单中找到`电视`按钮，点击`电视`按钮进入电视直播页面。
3. 在电视直播页面左侧会有电视频道列表，点击想要看的频道即可观看。
